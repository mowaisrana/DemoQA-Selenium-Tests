# DemoQA-Selenium-Tests
This repository contains automated UI tests for [DemoQA](https://demoqa.com) using **Selenium WebDriver** with **C#** and **MSTest/NUnit**.

## 🚀 Features
- Written in **C#** with Selenium WebDriver
- Uses **MSTest Unit Testing framework**
- Covers multiple modules of DemoQA:
  - ✅ Elements  
  - ✅ Forms (Practice Form automation)  
- Includes assertions to verify correct navigation and form submission  
- File upload, date picker, dropdowns, modals, and alert handling  

## 🛠️ Tech Stack
- **Language:** C#  
- **Automation Framework:** Selenium WebDriver  
- **Testing Framework:** MSTest 
- **Browser:** Chrome  

## 📂 Project Structure
Selenium_Pract2/
│-- UnitTest1.cs # Contains all test methods
│-- Drivers/ # WebDriver executables (ChromeDriver)
│-- TestData/ # Any test files (e.g., images for upload)
│-- README.md

## ▶️ How to Run Tests
1. Clone the repository:
   ```bash
   git clone https://github.com/<your-username>/DemoQA-Selenium-Tests.git
2. Open the solution in Visual Studio
3. Restore dependencies (NuGet: Selenium.WebDriver, Selenium.Support, MSTest)
4. Run tests using Test Explorer
